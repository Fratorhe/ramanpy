Post-processing Classes¶

We can either process a single file, or multiple files.

Multiple files¶

To read multiple files, we use the class ramanpy.ResultsDataFrames(). This class, internally uses ramanpy.ReadResultParamsFit() to read each result file. The idea is to have here everything needed to compute for example the intensity ratio.

compute_equivalent_La(Lambda=532, file_to_save=None)[source]¶

Computes the equivalent La from the formula of Cancado 2006 La = (2.4*10**(-10))*Lambda**4*(I_D/I_G)**-1 the laser wavelength is typically 532 nm The result is also in nm. WARNING: this method considers that this one compute_intensity_ratio_each_sample has been run before.

  • file_to_save – filename of to save to

Single file¶

To read the results of a single file, we use the class ramanpy.ReadResultParamsFit(). This should work for both Raman and XRD. However, most of the time we will process several files at once so this is not really important.
